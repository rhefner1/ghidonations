GHI Donations
=============

GHI Donations organizes the donations of Global Hope India (http://globalhopeindia.org) and was engineered by Ryan Hefner (http://rhefner.com).

It utilizes:

 - Google App Engine (Python 2.7)
 - App Engine NDB and webapp2 modules
 - Search API
 - JQuery frontend
 - And much more...
